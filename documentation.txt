Pet Nutrition Ecommerce Website Documentation

Intro:
Pet Nutrition ecommerce website is create using Django framework. The website will serve the following purpose.
    1. Landing Page
    2. Shop Page showcasing all the products
    3. Cart to checkout out the items.
    4. Login and maintaining user related profile.
